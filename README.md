netBoard is a community-first networking platform built to give HBCU students direct, personal access to local Black-owned businesses and researchers — no mass applications, no corporate noise, just real one-on-one connections.
Students can browse and filter opportunities by type, location, and pay, save listings, favorite companies, submit personal projects for collaboration, and build a profile visible to employers.
Employers can post opportunities, manage their team, and track student engagement with their listings.
Stack: React (Vite) · Java or Python backend (in progress) · MySQL · CSS custom properties with full dark/light mode support
Status: Frontend complete. Backend and database integration in progress.
